PMID to PMCID to ENA Accessions

Enter one PMID, PubMed URL, or PMC URL per line. If an article has a PMCID, full text is fetched from PMC and searched for ENA accession numbers.

No query run yet.

What This Script Does

  • Accepts one input per line: numeric PMID, PubMed URL, or PMC URL, then normalizes each line for lookup.
  • Resolves the normalized identifier to a PMC record, then verifies PMCID context.
  • If linked, resolves the PMCID and fetches full-text XML from PMC.
  • Runs ENA accession regex patterns over the full text and reports unique matches.

How to interpret outputs

ENA accession format reference: ENA accession documentation

Case 1: Not a PMC article
Status: This is not a PMCID article.
Meaning: The PMID does not have a `pubmed_pmc` link, so no full-text fetch is attempted.
Case 2: PMC article but no accession codes
Status: PMCID found (...). No ENA accession matches found.
Meaning: Full text is available, but none of the current ENA regex patterns matched text in this paper.
Case 3: PMC article with accession codes
Status: PMCID found (...). N accession(s) matched.
Meaning: One or more ENA-style accession strings were detected and listed in the output panel.
